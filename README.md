# Lazencas-HerosOfDestiny-GUI-
## 예전에 자바로 만들었던 카드플레이 방식의 RPG 스타일 게임 HerosOfDestiny의 CLI환경을 GUI환경으로 만들어 봤었다. 이떄당시 이미지 가공하는데에 많은 시간이 소모됬던게 기억난다. 게임의 이미지와 플레이 방식은 slay the spire를 많이 참조 했다.
